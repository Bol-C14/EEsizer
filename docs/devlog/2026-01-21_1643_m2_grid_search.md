## Rationale & notes

- GridSearchStrategy uses Patch + Guards + Operator pipeline for deterministic, reproducible exploration.
- Candidate lists, top-k, Pareto front, and a summary report are written into the run directory and indexed in the manifest.
- Coordinate mode is the default to avoid factorial blow-up; factorial mode is available for small spaces.
